Waypoint – Property Listing & Discovery Platform

Waypoint is a full-stack web application that allows users to discover, list, and manage rental properties across different destinations. The platform provides a smooth and secure experience for both property owners and travelers, with powerful filtering, search, and management features.



Key Features :

Authentication – Secure user sign-up and login system

Authorization – Only the listing owner can edit or delete their listings

Image Upload with Cloudinary – Upload and manage listing images efficiently

Category-Based Filtering – Browse listings by categories such as Trending, Rooms, Iconic Cities, Mountains, etc.

Search Functionality – Search listings by destination or country

Fully Responsive UI – Optimized for mobile, tablet, and desktop screens

Error Handling & Custom No-Results Page – User-friendly error messages and empty-state handling

Database Seeding – Preloaded sample listings for development and testing

File Upload & Editing Logic – Update listing details and images seamlessly




Tech Stack :

Frontend: EJS, HTML, CSS, JavaScript, Bootstrap

Backend: Node.js, Express.js

Database: MongoDB with Mongoose

Authentication: Passport.js

Image Storage: Cloudinary

Deployment: Render





Project Highlights :

Clean MVC architecture

Secure handling of environment variables

RESTful routes for listings, users, and reviews

Scalable and production-ready structure
